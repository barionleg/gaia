# GAIA visualization

### Running locally

1) Download data files: `data/download_tgas.sh`

2) Compress the data points so that they can render in the browser: `python convert.py`

3) Start a server: `python -m SimpleHTTPServer`

4) Visit http://localhost:8000

### Requirements

Python 2.7
